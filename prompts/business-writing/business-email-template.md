# ビジネスメール作成プロンプト

## 目的
簡潔で礼儀正しいビジネスメールを効率的に作成する

## 使い方
1. 以下のプロンプトをAIに入力
2. [ ]内の項目を自分の状況に合わせて記入
3. 生成されたメールを確認・調整

## プロンプト例

```
以下の情報に基づいて、ビジネスメールを作成してください:

【宛先】: [相手の名前・役職]
【目的】: [メールの目的を簡潔に]
【背景】: [必要に応じて状況を説明]
【依頼・提案内容】: [具体的な内容]
【希望返信期限】: [期限があれば]

【トーン】: 
- [ ] フォーマル（初めての相手、重要な案件）
- [ ] 標準（通常のビジネス）
- [ ] カジュアル（社内、親しい取引先）

条件:
- 件名も提案してください
- 簡潔で分かりやすい文章にしてください
- 相手の立場に配慮した表現を使ってください
```

## 期待される出力

以下のような構成のメールが生成されます:

```
件名: [適切な件名]

[宛名]様

お世話になっております。[あなたの名前]です。

[導入部分]
[本文]
[依頼・提案]
[締めの言葉]

何卒よろしくお願いいたします。

[署名]
```

## カスタマイズ例

### より具体的な指示を追加する場合:
```
追加条件:
- 文字数は300字以内
- 箇条書きで要点を整理
- 添付ファイルへの言及を含める
```

## 注意事項

- 生成されたメールは必ず自分で確認・調整してください
- 機密情報や個人情報は慎重に扱ってください
- 相手との関係性や文化的背景も考慮してください
