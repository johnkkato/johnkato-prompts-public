# 🤝 Contributing to johnkato-prompts-public

このリポジトリへの貢献に興味を持っていただきありがとうございます！

## 📝 貢献方法

### 1. 新しいプロンプトの提案
[Discussions](https://github.com/johnkato/johnkato-prompts-public/discussions) で新しいプロンプトのアイデアを共有してください。

### 2. 既存プロンプトの改善
1. このリポジトリをFork
2. 新しいブランチを作成: `git checkout -b improve-prompt-name`
3. 変更をコミット: `git commit -m "Improve: プロンプト名の改善"`
4. Pushする: `git push origin improve-prompt-name`
5. Pull Requestを作成

### 3. バグ報告・質問
[Issues](https://github.com/johnkato/johnkato-prompts-public/issues) または [Q&A](https://github.com/johnkato/johnkato-prompts-public/discussions/categories/q-a) をご利用ください。

## ✅ プロンプト投稿ガイドライン

新しいプロンプトを投稿する際は、以下の形式に従ってください:

```markdown
# プロンプトのタイトル

## 目的
このプロンプトが解決する課題を簡潔に説明

## 使い方
1. ステップ1
2. ステップ2

## プロンプト例
\```
[ここにプロンプトの内容]
\```

## 期待される出力
どのような結果が得られるか

## 注意事項
- 制限事項や注意点があれば記載
```

## 🚫 禁止事項

- 個人情報、機密情報を含むコンテンツ
- 他者の著作権を侵害するコンテンツ
- 有害、差別的、違法なコンテンツ

## 📜 ライセンス

投稿されたコンテンツは [CC-BY-4.0](LICENSE) ライセンスの下で公開されることに同意したものとみなします。
